## 🏗️ AUTO-GENERATED FILES

These files are automatically created during the build process:

15. **lex.yy.c** (generated by Flex from jujutsu.l)
    - C scanner implementation
    - Used by Bison during compilation
    - Auto-created: `make parser`
    - Auto-cleaned: `make clean`

16. **jujutsu.tab.c** (generated by Bison from jujutsu.y)
    - C parser implementation
    - Contains parsing logic
    - Auto-created: `make parser`
    - Auto-cleaned: `make clean`

17. **jujutsu.tab.h** (generated by Bison from jujutsu.y)
    - Parser header with token definitions
    - Included by jujutsu.l
    - Auto-created: `make parser`
    - Auto-cleaned: `make clean`

18. **jujutsu** (executable)
    - **Type**: Compiled executable
    - **Created by**: g++ compilation
    - **Status**: ✅ Ready to run
    - **Size**: 84 KB
    - **Usage**: `./jujutsu <input.jjk>`
